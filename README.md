## 🚀 Sobre mí

Soy **TSU en Desarrollo de Software Multiplataforma**, enfocado en la creación de soluciones web funcionales, sistemas a medida y automatizaciones inteligentes para negocios.

Me especializo en combinar **desarrollo web con Laravel** y **automatización de procesos con n8n**, integrando herramientas como APIs, bases de datos, flujos automatizados e inteligencia artificial para resolver problemas reales.

Mi enfoque está en construir tecnología que no solo se vea bien, sino que ayude a ahorrar tiempo, ordenar procesos y convertir ideas en productos digitales útiles.
